# Post-Malone-Data-Analysis
Tento projekt analyzuje diskografii Post Malona (2016–2024) na YouTube Music. Pomocí SQL a Power BI jsem zpracoval 21,9 mld. streamů a vypočítal metriky jako „Hustota hitů“ či „Dominance singlu“. Výsledkem je hloubková analýza vztahu mezi délkou alb a úspěchem virálních hitů.
